# IronCore-Fitness-Gym-Website
A responsive fitness gym website with smooth scroll animations, phone number validation, and a form that saves user details directly to Google Sheets.

---

## 🌐 Live Demo
[Click here to view the live dashboard](https://app.netlify.com/projects/ironcore-fitness-gym/overview)

---

## 🚀 Key Features

- **Lead Generation System:** Built a contact form that sends user details directly to **Google Sheets** in real-time.  
- **Phone Number Validation:** Used `intl-tel-input` to handle international formats with country codes and proper 10-digit validation.  
- **Modern UI/UX:** Designed a clean dark-themed interface using HTML5 and CSS3.  
- **Scroll Animations:** Elements appear smoothly on scroll to make the website more interactive.  
- **Mobile-First Design:** Optimized layout and drawer menu for a smooth experience on mobile devices.  

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)  
- **Validation:** International Telephone Input (JS Library)  
- **Backend/Data Handling:** Google Apps Script with Google Sheets  
---

## ⚙️ Core Concepts Used

- **Asynchronous Data Handling:** Used `fetch()` to send form data without reloading the page.  
- **DOM Manipulation:** Handled menu toggle, overlay, and animations using JavaScript.  
- **Input Validation:** Applied regex and logic to ensure correct phone number and user input.
